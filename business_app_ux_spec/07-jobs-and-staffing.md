# 7. Jobs and Staffing

# Jobs tab inside a project

## Combined planned and scheduled view

Show both:

- Planned but unscheduled work
- Scheduled jobs

Unscheduled work must remain visible.

## Grouping

For moving projects:

- אריזה
- פריקה

For standalone projects:

- אריזה
- פריקה
- סידור

## Planned-service card

Example:

```text
אריזה

משוער:
2 ימי עבודה
4 עובדים בכל יום

נקבע:
יום עבודה אחד

נותר:
יום עבודה אחד

[קביעת יום נוסף]
```

## Scheduled-job card

Show:

- Date
- Time
- Address
- Required workers
- Assigned workers
- Manager state
- Publication state
- Job state
- Forms state
- Attendance state

## Job actions

- Open
- Edit
- Copy
- Cancel
- Publish
- Unpublish
- Assign workers
- Send worker reminder
- Review attendance
- Add another day

# Job detail page

## Header

Show:

- Project/customer
- Job type
- Date
- Time
- Address
- Project status
- Job status

Contextual primary actions:

- Publish job
- Assign workers
- Start job
- Review attendance
- Complete job

## Tabs

- פרטי עבודה
- עובדים
- נוכחות
- טפסים
- הערות

## Job details

Show:

- Date and time
- Expected duration
- Address
- Contact person
- Required workers
- Manager requirements
- Worker instructions
- Internal notes
- Publication state
- Customer-form state

Actions:

- Edit
- Copy
- Cancel
- Open project
- Contact customer
- Navigate

# Staffing

## Slot-based display

Show manager and worker positions separately.

Example:

```text
מנהל עבודה

דנה לוי
מאושרת
```

```text
עובדים

יוסי כהן
מאושר

נועה לוי
מאושרת

מקום פנוי

מקום פנוי
```

## Staffing states

Show:

- Required positions
- Filled positions
- Pending requests
- Reserved manager slots
- Waiting list
- Worker shortage
- Manager shortage

## Actions

- Assign worker
- Approve request
- Reject request
- Remove worker
- Contact worker
- Promote from waiting list
- Request replacement
- Publish remaining positions

## Readiness checklist

Show:

```text
מוכנות לעבודה
```

Checklist:

- Project approved
- Customer details complete
- Address confirmed
- Enough workers
- Manager assigned
- Customer form complete
- Worker instructions sent

## Publication

Job publication states:

- Draft
- Published
- Fully staffed
- In progress
- Awaiting attendance review
- Completed
- Cancelled
