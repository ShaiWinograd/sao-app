# 9. Customers and Workers

# Customers page

## Customer list

Show:

- Name
- Phone
- Location
- Active projects
- Last project
- Total projects
- Outstanding-payment indicator

## Customer detail

Tabs:

- פרטים
- פרויקטים
- הודעות
- הערות

Show:

- Contact details
- Preferred communication
- Addresses
- Notes
- Active projects
- Completed projects
- Payment issues
- Forms and communication history

## Duplicate prevention

Search potential duplicates by:

- Phone
- Email
- Name

Warning:

```text
ייתכן שהלקוח כבר קיים במערכת
```

Actions:

- Use existing customer
- Review matches
- Create anyway

# Workers page

## Worker list

Show:

- Worker name
- Role
- Active/inactive
- Manager eligibility
- Upcoming jobs
- Availability
- Missing forms
- Attendance issues

## Worker detail tabs

- פרטים
- זמינות
- עבודות
- נוכחות
- תשלומים

## Worker profile

Show:

- Contact information
- Worker type
- Eligible job types
- Manager eligibility
- Hourly wage
- Fixed daily payment
- Bonuses
- Active status
- Notes

## Worker calendar

Show:

- Assigned jobs
- Requested jobs
- Unavailable dates
- Confirmed availability
- Conflicts

## Worker privacy

Workers cannot see:

- Customer pricing
- Other workers' wages
- Quotation details
- Customer payment status
- Internal notes
- Other workers' personal details
